A very basic chat-bot that can find Quotes for you.

Ping any keyword on our FB page's Chat to get back a random Quote instantly.

Fork On Github Go to BOT >

How it works?

It's a very basic chat-bot built using Facebook Messenger API and a Node JS webhook to textmine www.brainyquote.com and find matching quotes. Once found it pings you back a random one from the results.
